Belajar CSS

Apa itu CSS?

CSS atau Cascading Style Sheet adalah dokumen web yang memiliki fungsi mengatur tampilan web dokumen lain, yaitu HTML. Beberapa contoh pemanfaatan CSS pada HTML adalah untuk mengatur warna latar belakang, gaya huruF, jarak antar elemen, dan masih banyak lagi.
CSS terdiri atas kumpulan kode-kode yang keberadaannya tidak menggantikan HTML menjadi lebih indah dan dinamis. Berikut ini ada contoh dokumen CSS.

Bagaimana menggunakan CSS?

Ada 3 cara menggunakan CSS:

  1. Inline Styles: Menuliskan kode CSS langsung pada tag yang akan dirubah atau diatur style-nya
  2. Internal/Embedded Styles: Menuliskan kode CSS pada bagian header dokumen HTML
  3. Eksternal/Linked Styles: Menuliskan kode CSS pada file terpisah dan dipanggil melalui tag link dari dokumen HTML
